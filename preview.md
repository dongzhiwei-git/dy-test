# 抖音风格聊天应用 - 预览说明

## 🎨 设计亮点

### 1. 顶部状态栏
- 黑色背景，仿iPhone刘海屏设计
- 左侧显示时间（21:17）
- 中间有刘海（notch）区域
- 右侧显示信号和电池图标

### 2. 导航栏
- 白色背景，圆形头像
- 支持点击头像更换
- 视频通话和更多选项按钮

### 3. 消息气泡
- **左侧消息**：白色气泡，左上角圆角较小
- **右侧消息**：蓝色渐变气泡（#4A9FF5 → #5B7EF6），右上角圆角较小
- 圆润的设计，带阴影效果
- 圆形头像设计

### 4. 底部输入区
- 白色背景，带顶部阴影
- 表情快捷栏：白色圆角按钮
- 输入框：灰色背景，圆角设计
- 图标按钮：hover效果

### 5. 特色功能面板
- **更多功能面板**：白色背景，顶部圆角，阴影效果
- **语音录制面板**：深色半透明背景，毛玻璃效果，蓝色渐变波形
- **表情选择器**：白色背景，顶部圆角

## 🚀 快速开始

1. 打开 `index.html` 文件
2. 在输入框输入消息并发送
3. 点击任意头像切换发送用户
4. 使用底部功能按钮体验更多功能

## 🎮 交互说明

- **发送消息**：输入框 + 回车键
- **切换用户**：点击消息头像
- **发送图片**：+ 按钮 → 照片
- **语音录制**：麦克风按钮
- **更换头像**：+ 按钮 → 更换头像
- **快捷表情**：点击底部表情按钮

## 🌈 配色方案

- **主题色**：蓝色渐变（#4A9FF5 → #5B7EF6）
- **背景色**：#F5F5F5（浅灰）
- **文字色**：#161823（深色）
- **辅助色**：#8A8A8A（灰色文字）
- **白色**：#FFFFFF（卡片、气泡）

## 💡 技术特点

- 纯前端实现，无需后端
- 响应式设计，支持移动端
- CSS3 动画和渐变
- 现代化的 UI 设计
- 流畅的交互体验

# 使用指南

## 🚀 快速开始

### 方式1：直接打开（最简单）
双击 `index.html` 文件，在浏览器中直接打开即可使用。

### 方式2：本地服务器（推荐，支持语音功能）
```bash
# 在项目目录打开终端，运行以下命令之一：

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (需要安装 http-server)
npx http-server -p 8000
```

然后在浏览器访问：`http://localhost:8000`

## 📱 功能使用

### 1️⃣ 发送文字消息
- 在底部输入框输入文字
- 按 **Enter** 键发送
- 或者点击发送按钮

### 2️⃣ 切换发送用户（模拟双人聊天）
- 点击**任意消息的头像**
- 会自动切换当前发送用户（左侧/右侧）
- 会弹出提示："已切换到左侧用户" 或 "已切换到右侧用户"

### 3️⃣ 发送图片
1. 点击底部的 **+** 按钮
2. 选择 **"照片"** 选项
3. 从电脑选择图片文件
4. 图片会自动发送到聊天窗口

### 4️⃣ 录制语音消息
1. 点击底部的 **🎤 麦克风**按钮
2. 浏览器会请求麦克风权限，点击**允许**
3. 开始录音，会显示录音时长
4. 点击 **"发送"** 完成录音
5. 或点击 **"取消"** 放弃本次录音

⚠️ **注意**：语音功能需要：
- 使用本地服务器（localhost）或 HTTPS
- 授予浏览器麦克风权限

### 5️⃣ 更换头像

#### 更换自己的头像（当前发送用户）
1. 点击底部的 **+** 按钮
2. 选择 **"更换头像"** 选项
3. 选择图片文件

#### 更换聊天对象的头像
1. 点击**顶部导航栏的圆形头像**
2. 选择图片文件

### 6️⃣ 使用表情

#### 方式1：快捷表情
点击底部的快捷表情按钮：
- 👋 晚上好
- 😘 比个心
- 👍 赞
- 😂 捂脸
- 🌹 玫瑰

#### 方式2：表情选择器
1. 点击底部的 **😊 笑脸**按钮
2. 从表情面板选择表情
3. 表情会插入到输入框

### 7️⃣ 更多功能
点击底部的 **+** 按钮，可以访问：
- 📷 照片
- 📸 拍摄
- 👤 更换头像
- 📍 位置
- 🎤 语音输入
- 👥 视频通话

## 🎨 界面说明

### 顶部状态栏
- **时间**：显示当前时间（自动更新）
- **刘海**：仿iPhone刘海屏设计
- **信号/电池**：装饰性图标

### 导航栏
- **返回按钮**：点击返回上一页
- **聊天人数**：显示"2"表示双人聊天
- **头像+名字**：点击头像可更换对方头像
- **视频通话**：视频通话按钮（装饰）
- **更多选项**：更多菜单（装饰）

### 消息区域
- **左侧消息**：白色气泡，对方发送
- **右侧消息**：蓝色渐变气泡，自己发送
- **时间戳**：每10条消息自动添加
- **系统消息**：灰色居中显示

### 底部输入区
- **表情快捷栏**：常用表情快速发送
- **网格按钮**：打开更多功能
- **输入框**：输入文字消息
- **语音按钮**：录制语音消息
- **表情按钮**：打开表情选择器
- **加号按钮**：打开更多功能面板

## 💡 使用技巧

### 模拟真实对话
1. 输入消息 → 发送（作为右侧用户）
2. 点击任意头像切换到左侧用户
3. 输入回复 → 发送（作为左侧用户）
4. 重复以上步骤，实现双人对话

### 个性化设置
- 分别为左右两个用户设置不同的头像
- 点击消息头像切换身份
- 使用表情让对话更生动

### 查看大图
- 点击聊天中的图片可以在新标签页查看大图

## 🔧 常见问题

### Q: 语音功能无法使用？
**A:** 
1. 确保使用本地服务器（localhost）或 HTTPS
2. 检查浏览器麦克风权限
3. Chrome/Edge浏览器支持最好

### Q: 刷新后消息丢失？
**A:** 
这是正常现象，所有消息存储在内存中。如需保存消息，可以：
- 截图保存
- 后续版本可添加本地存储功能

### Q: 如何区分两个用户？
**A:** 
- **左侧消息**（白色气泡）= 对方
- **右侧消息**（蓝色渐变气泡）= 自己
- 点击任意消息头像可切换当前发送身份

### Q: 移动端支持吗？
**A:** 
完全支持！界面采用响应式设计，在手机上也能完美显示。

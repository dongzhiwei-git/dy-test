# 微信聊天界面克隆

一个高度还原的微信聊天界面前端项目，支持完整的聊天功能。

## 功能特性

### ✅ 已实现功能

1. **聊天功能**
   - 发送文字消息
   - 发送表情符号
   - 发送图片（支持本地上传）
   - 发送语音消息（模拟录音）
   - 消息气泡样式（左右分布）
   - 消息编号显示
   - 输入中状态提示
   - 自动回复功能

2. **用户界面**
   - 修改聊天对象头像
   - 修改聊天对象名字
   - 头像在所有消息中同步更新

3. **交互功能**
   - 快捷表情反应栏
   - 表情面板选择器
   - 加号菜单（照片、拍摄、修改头像、修改名字）
   - 长按消息撤回（自己的消息）
   - 语音录制界面
   - 消息滚动动画

4. **界面设计**
   - 完全还原微信聊天界面样式
   - 响应式设计，支持移动端
   - 流畅的动画效果
   - 精确的图标和配色

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件

2. 或使用本地服务器：
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

然后访问 `http://localhost:8000`

## 功能说明

### 发送消息
- 在底部输入框输入文字，按回车发送
- 点击快捷表情按钮快速发送表情
- 点击笑脸图标打开表情面板

### 发送图片
1. 点击底部加号按钮
2. 选择"照片"
3. 选择要发送的图片文件

### 发送语音
1. 点击麦克风图标开始录音
2. 录音中会显示计时器和波形动画
3. 点击"发送"发送语音，或"取消"取消录音

### 修改头像
1. 点击底部加号按钮
2. 选择"修改头像"
3. 选择新的头像图片

### 修改名字
1. 点击底部加号按钮
2. 选择"修改名字"
3. 输入新的名字

### 撤回消息
- 长按自己发送的消息（1秒）
- 在弹出的确认框中选择确定

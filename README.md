## 🚀 快速开始

### 方法一：直接打开
直接在浏览器中打开 `douyin.html` 文件即可！

### 方法二：本地服务器
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000/douyin.html`

## 📖 使用说明

### 💬 发送消息
1. 在底部输入框输入文字
2. 按 **Enter** 键发送
3. 或点击快捷表情按钮快速发送

### 😊 使用表情
- **快捷表情**：点击底部快捷表情栏的按钮
- **完整表情面板**：点击笑脸图标打开表情选择器

### 🎤 发送语音
1. 点击麦克风图标
2. 开始录音（会显示计时和波形）
3. 点击"发送"完成，或"取消"放弃

### 📷 发送图片
1. 点击加号按钮（➕）
2. 选择"照片"
3. 从本地选择图片文件

### 👤 修改资料
1. 点击加号按钮（➕）
2. 选择"修改头像"或"修改名字"
3. 上传新头像或输入新名字

### ↩️ 撤回消息
- **长按**自己发送的消息 1 秒
- 在弹出的确认框中选择"确定"

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| Enter | 发送消息 |
| ESC | 关闭弹出面板 |
| 长按 1秒 | 撤回消息 |

## 🔧 自定义配置

### 修改配色
在 `douyin.css` 中修改以下变量：

```css
/* 发送消息气泡颜色 */
.message-sent .message-bubble {
    background: #5B9EFF;  /* 修改这里 */
}

/* 接收消息气泡颜色 */
.message-bubble {
    background: #ffffff;  /* 修改这里 */
}
```

### 修改头像
在 `douyin.js` 中修改：

```javascript
let contactAvatar = 'https://via.placeholder.com/40/FF6B9D/ffffff?text=TA';
let selfAvatar = 'https://via.placeholder.com/40/00D9FF/ffffff?text=我';
```
